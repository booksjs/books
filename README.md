## Using BooksJS

Link PDF.js in your project, use this CDN:
+ https://cdnjs.com/libraries/pdf.js

After PDF.js link BooksJS in your project using this CDN:
+ https://booksjs.github.io/books/js/books.min.js

#### You are good to go!
